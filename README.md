# Sistema de Reserva de Assentos (Java)

Projeto desenvolvido para prática de:

- Arrays
- Listas (ArrayList)
- Estruturas de repetição
- Validações
- Menu interativo no console
- Lógica de programação

---

## Funcionalidades

✔ Reservar assento  
✔ Consultar assentos vagos  
✔ Consultar assentos ocupados  
✔ Cancelar reserva  
✔ Validação de assentos (1–20)  
✔ Verificação de assento já ocupado

---

## Tecnologias utilizadas

- Java
- IntelliJ IDEA
- Git
- GitHub
